IBM i Playbook Examples
=================
This folder contains examples and best practices for building Ansible Playbooks for IBM i.
1. To install yum to IBM i nodes, refer to ibmi-install-yum.yml
2. To enable non-password signon from Ansible server to IBM i nodes, refer to ssh-addkey.yml
3. About how to use ibmi_cl_command module, refer to ibmi-cl-command-sample.yml
4. About how to use ibmi_sql_execute module or ibmi_sql_query module, refer to ibmi-sql-sample.yml
